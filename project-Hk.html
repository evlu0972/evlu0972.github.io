Health Keeper APP

portfolio / Project

amor Health series

AI運算智慧管理替自己的健康把關

Health Keeper是一款結合自我健康管理概念的應用程式。可搭配amor健康系列的裝置或直接使用手機來記錄，將透過手機功能獲取步數、卡路里、心率等數值計算。除此之外，你還可以輸入自己的生理數值，如血壓、血糖、體重、體溫等。

在收集用戶的各種量測數據後，經人工智慧進行分析以獲得結果，圖表將以月、周和日區間顯示。你可以透過追蹤記錄來實現自我健康管理。

設計挑戰-1

可自由排列顯示自己關注的資訊

APP首面總覽呈現各項目數值顯示，如何讓用戶清楚掌握自己所心關心的項目數值。


設計策略-1

總覽設計不同類型排列分別為列表和磚塊方式呈現資訊，用戶可以自由拖移列表或磚塊順序排列位置及開關顯示項目。

經使用者問卷調查結果及回饋，版型B和D各取其優點延伸設計，最後數值呈現以大磚塊將icon和資料對齊中，讓用戶可集中聚焦視線。

管理項目顯示 / APP總覽表 / 無任何項目






設計挑戰-2

各類圖表彙整，一目了然

各項數值的標準高低標不同，如何使各類圖表達成畫面統一性顯示及讓用戶快速地掌握資訊。


設計策略-2

將項目數值做進一步歸納整理分類，經和UX 討論後，定義出各項目的縱軸高低標、計算規則、顯示範圍及超標或過低標時要如何呈現曲線圖。

各類項目圖表 / 活動量 / 身體數值 / 疲勞指數 / 循環指數 / 睡眠分析




UI design

森林綠搭配海水及天空藍，讓大自然形象融入介面中

Health Keeper 配色以健康、適合各年齡層的使用者為理念素求，擷取以大自然形象的森林綠為介面主系色，再搭天空和海水藍、太陽黃、及珊瑚紅、淺玫瑰紅等配色來點綴使整體介面層次感更加鮮明。







APP設計亮點 -1

達成目標給予不同視覺回饋

活動項目數值以淺藍、深藍圓形bar 代表未達成預設目標，當圓形bar 轉變成綠色時，恭禧你! 達成活動目標並給予不同視覺呈現以及體重達到目標時也會出現全畫面動畫來通知用戶。








APP設計亮點 -2

數值異常即時發現

當身體生理數值高出或低於標準值時，異常數值和圖示則會轉變成珊瑚紅，用來提醒用戶需注意身體狀況及定時追蹤。
